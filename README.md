# saramalvar.github.io

Personal website of **Sara Malvar, Ph.D.** — AI Engineer (Generative AI, LLMs, copilots).

A hand-built static site (HTML / CSS / vanilla JS), served directly from the `main` branch via GitHub Pages — no build step.

## Structure
- `index.html` — all content (single page)
- `styles.css` — styling, dark/light themes, blue accent
- `script.js` — theme toggle, mobile menu, scroll reveal
- `assets/` — images (`prof_pic.jpg`, optional `cv.pdf`)

## Edit
Just edit `index.html` and push to `main`. The site updates automatically.

## To personalize
- Add your photo at `assets/prof_pic.jpg`
- Add your CV at `assets/cv.pdf` (enables the download button)
